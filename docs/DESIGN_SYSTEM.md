# LiteMark 设计系统

本文档定义了 LiteMark 水印模板的设计规范和视觉原则。

## 设计原则

1. **不喧宾夺主**：水印是照片的配角，不应抢夺视觉焦点
2. **信息层级清晰**：通过字重、字号、颜色区分信息重要性
3. **比例自适应**：所有尺寸基于照片分辨率的比例计算，任意尺寸都美观
4. **风格多样**：不同场景适配不同渲染模式

---

## 间距系统

基于 **4px 网格** 的间距节奏：

```
基础单位: 4px
├── 小型间距:  8px  (2×)
├── 标准间距:  16px (4×)
├── 中型间距:  24px (6×)
└── 大型间距:  32px (8×)
```

### 模板中的间距表达

所有间距通过 `padding_ratio` 和 `frame_height_ratio` 间接控制：

```
相框高度 = 短边 × frame_height_ratio
实际间距 = 相框高度 × padding_ratio
```

**推荐组合：**

| 风格 | frame_height_ratio | padding_ratio | 视觉感受 |
|------|-------------------|---------------|----------|
| 极简 | 0.06 | 0.08 | 紧凑、克制 |
| 标准 | 0.10 | 0.10 | 平衡、舒适 |
| 专业 | 0.12 | 0.12 | 宽松、大气 |

## 颜色系统

### 基础色板

```
主文字色    #1A1A1A   (深黑，比纯黑柔和)
次文字色    #4A4A4A   (中深灰)
辅助文字色  #6A6A6A   (中灰，参数信息)
浅色文字    #FFFFFF   (纯白)
半透明文字  #CCCCCC   (浅灰)

分隔线色    #E8E8E8   (极淡灰，不刺眼)
```

### 视觉层级配色

```
Level 1 (最重要): #1A1A1A  bold  → 摄影师名
Level 2 (重要):   #4A4A4A  normal → 相机·镜头
Level 3 (次要):   #6A6A6A  normal → 光圈·快门·ISO·焦距
Level 4 (装饰):   #E8E8E8  —      → 分隔线
```

### Alpha 通道使用

Overlay 模式的背景半透明：
```
暗底背景:   #00000080  (黑色 50%)
浅底背景:   #FFFFFFB3  (白色 70%)
```

GradientFrame 的渐变遮罩：
```
顶部:  rgba(255, 255, 255, 0.25)  (25% 白)
底部:  rgba(255, 255, 255, 1.0)   (100% 白)
```

## 字体系统

### 字重规范

| 字重 | 用途 | 回退策略 |
|------|------|----------|
| **Bold** | 摄影师名、主标题 | 无 Bold 字体时回退 Regular |
| **Normal** | 相机型号、参数行 | 默认字重 |
| **Light** | 日期、极简风格的装饰文字 | 无 Light 时回退 Normal |

### 字号比例

所有字号基于 **相框高度** 的比例：

```
主字号  = 相框高度 × primary_font_ratio   (推荐 0.20 - 0.28)
次字号  = 相框高度 × secondary_font_ratio (推荐 0.14 - 0.20)
```

### 行高

默认行高 = 字号 × 1.4（自动计算）

## 渲染模式设计规范

### Bottom Frame（底部白框）

```
┌─────────────────────────────┐
│                             │
│         原始照片             │
│                             │
├─────────────────────────────┤ ← 清晰边界
│  作者                    参数 │
│  相机 · 镜头              焦距 │
└─────────────────────────────┘
```

- **背景**：纯白 `#FFFFFF`
- **文字**：深色层级 `#1A1A1A` → `#4A4A4A` → `#6A6A6A`
- **分隔**：竖线 `#E8E8E8`
- **适用**：正式展示、作品集

### Gradient Frame（渐变过渡）

```
┌─────────────────────────────┐
│                             │
│         原始照片             │
│░░░░░░░░░░░░░░░░░░░░░░░░░░░░░│ ← 25% 白渐变
│░░░░░░░░░░░░░░░░░░░░░░░░░░░░░│ ← 50% 白渐变
│░░░░░░░░░░░░░░░░░░░░░░░░░░░░░│ ← 100% 白
│  作者                    参数 │
└─────────────────────────────┘
```

- **背景**：顶部透明 → 底部纯白（线性渐变）
- **文字**：同 Bottom Frame
- **分隔**：无竖线，用留白分隔
- **适用**：风景摄影、现代风格

### Minimal（极简线条）

```
┌─────────────────────────────┐
│                             │
│         原始照片             │
│                             │
─────────────────────────────── ← 1px 细线
│  作者 · 相机 · 光圈 · ISO    │
└─────────────────────────────┘
```

- **背景**：无（透明）
- **文字**：单行，用 `·` 连接
- **分隔**：底部 1px 细线 `#E8E8E8`
- **适用**：极简主义、社交媒体

### Overlay（照片内嵌）

```
┌─────────────────────────────┐
│                             │
│         原始照片             │
│                   ┌────────┐│
│                   │ 作者   ││ ← 半透明暗底
│                   │ 参数   ││   圆角矩形
│                   └────────┘│
└─────────────────────────────┘
```

- **背景**：不扩展画布，右下角圆角矩形 `#00000080`
- **文字**：纯白 `#FFFFFF`
- **圆角**：半径 12px
- **内边距**：16px
- **适用**：不想裁切照片、签名风格

## Logo 使用规范

### 尺寸

```
Logo 高度 = 相框高度 × logo_size_ratio (推荐 0.35)
```

### 位置

- 默认位于左栏和右栏之间
- 保持原始纵横比
- 双线性插值缩放（消除锯齿）

### 支持的格式

- PNG（推荐，支持透明背景）
- JPEG
- WebP

---

## 响应式设计

LiteMark 的模板系统天然支持响应式：

```
相框高度 ∝ 短边长度
字体大小 ∝ 相框高度
间距     ∝ 相框高度
```

这意味着同一份模板在 800×600 和 6000×4000 的照片上比例保持一致。

### 极端尺寸适配

- **极小图像** (< 400px 短边)：相框高度保底 80px
- **超大图像** (> 8000px 短边)：相框高度上限 20% 短边

## 设计检查清单

创建新模板时，请确认：

- [ ] 选择了合适的 `render_mode`
- [ ] 文字颜色与背景有足够对比度
- [ ] 使用了视觉层级（至少 2 级颜色深度）
- [ ] 信息不超过 4 行
- [ ] `frame_height_ratio` 在 0.06 - 0.15 之间
- [ ] 在深色和浅色照片上分别测试
- [ ] 在 800×600 和 1920×1080 上分别测试
